How to set up
  1. Open console.cloud.google.com and sign in
  2. Create a new project (or pick an existing one)
  3. Search for Google Calendar API and click Enable
  4. Go to Credentials in the left menu
  5. Click + Create Credentials, then choose OAuth client ID
  6. For Application type, pick Web application
  7. Scroll down to Authorised redirect URIs, click Add URI, and paste the URI shown above
  8. Click Create, then copy the Client ID and paste it below
